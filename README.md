💻 Installation
---

Be sure, you have installed all dependencies to the project.  

1️. Clone this repository

`git clone https://github.com/Aurelily/deliveroo-frontend.git`

`cd deliveroo-frontend`

2️. Install packages

`npm install`
or
`yarn`

3️. When installation is complete:

`yarn start`
